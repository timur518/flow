# Auth Sections - Компоненты авторизации

Этот каталог содержит компоненты для модального окна авторизации с тремя динамическими разделами:
1. **LoginForm** - Форма входа
2. **RegisterForm** - Форма регистрации
3. **ForgotPasswordForm** - Форма восстановления пароля

## Структура

```
auth-sections/
├── LoginForm.tsx           # Форма входа
├── RegisterForm.tsx        # Форма регистрации
├── ForgotPasswordForm.tsx  # Форма восстановления пароля
├── index.ts               # Экспорт всех компонентов
└── README.md              # Документация
```

## Использование

### AuthModal

Главный компонент модального окна авторизации с навигацией между разделами:

```tsx
import { AuthModal } from '@/components/modals';

function MyComponent() {
    const [authModalOpen, setAuthModalOpen] = useState(false);

    return (
        <AuthModal
            isOpen={authModalOpen}
            onClose={() => setAuthModalOpen(false)}
            initialSection="login" // 'login' | 'register' | 'forgot-password'
        />
    );
}
```

### Стили

Все стили для форм авторизации находятся в `resources/react/styles/popup.css`:

- `.auth-form` - Контейнер формы
- `.auth-form-fields` - Белый блок с полями (border-radius: 24px)
- `.auth-form-label` - Лейблы полей (12px semibold #737373)
- `.auth-form-input` - Поля ввода (#222222)
- `.auth-form-submit` - Кнопка отправки (стиль как у "Оформить заказ")
- `.auth-form-legal` - Юридическая информация (12px semibold #737373)
- `.auth-success-overlay` - Оверлей для уведомления об успехе

### Анимация

Переходы между компонентами используют анимацию слайда справа (как в ProfileModal):
- Вперед: `slide-left` (350ms cubic-bezier)
- Назад: `slide-right` (350ms cubic-bezier)

## Компоненты

### LoginForm

Форма входа с полями:
- Email или телефон
- Пароль (с кнопкой "Забыли пароль?")

Кнопки:
- "Войти" - отправка формы
- "Зарегистрироваться" - переход к регистрации
- "Забыли пароль?" - переход к восстановлению

### RegisterForm

Форма регистрации с полями:
- Имя
- Телефон (маска +7(999)999-99-99)
- Электронная почта
- Пароль
- Подтверждение пароля

Кнопки:
- Назад (стрелочка)
- "Зарегистрироваться" - отправка формы
- "Войти" - переход к входу

### ForgotPasswordForm

Форма восстановления пароля с полем:
- Электронная почта

Кнопки:
- Назад (стрелочка)
- "Восстановить пароль" - отправка формы

После успешной отправки показывается уведомление:
"Письмо со ссылкой на сброс пароля отправлено на вашу почту"
с кнопкой "Спасибо"

## API

Компоненты используют методы из `authService`:
- `login(data)` - Вход
- `register(data)` - Регистрация
- `forgotPassword(email)` - Запрос восстановления пароля

## Дизайн

Все компоненты следуют единому стилю:
- Белые блоки с закруглением 24px
- Лейблы: 12px semibold #737373
- Текст в полях: #222222
- Кнопка "Забыли пароль?": 10px semibold #FDAFC0
- Отступ между блоками: 25px
- Юридическая информация: 12px semibold #737373
- Кнопка отправки: стиль как у "Оформить заказ в корзине"
